## This project uses GNU Autotools and make for C++
### Building and Installing
To build and install run:
```
mkdir build && cd build
../configure
make && make install
```
To build and install locally run:
```
mkdir build && cd build
../configure --prefix=INSTALL/DIR/HERE
make && make install
```
